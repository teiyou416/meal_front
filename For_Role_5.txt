# AI Recommendation Page Development Guide
# 今日推荐页面开发指南

## English

You are responsible for the AI meal recommendation page.

Main file to edit:

src/pages/RecommendationView.vue

This page should display the AI-generated meal recommendation for the selected date. It may include recommended meals, reasons, and suggestions based on the user's activities, past meals, allergies, and dietary preferences.

You may need to use these files:

src/stores/date.ts
src/stores/user.ts
src/api/recommendation.ts
src/types/index.ts

Important existing state:

useDateStore

It provides:

selectedDate
selectedMonth
setSelectedDate()
goToday()

Important API function:

getRecommendation(date)

Example usage:

const { selectedDate } = storeToRefs(useDateStore())

const result = await getRecommendation(selectedDate.value)

Expected recommendation type:

MealRecommendation

Defined in:

src/types/index.ts

Recommended development scope:

1. Edit src/pages/RecommendationView.vue
2. Read selectedDate from src/stores/date.ts
3. Call getRecommendation(selectedDate.value)
4. Display the recommended meals and AI explanation
5. Add loading and error states
6. Do not modify request.ts unless the backend response format changes
7. Do not modify router unless you need a new page

## 中文

你负责开发 AI 今日推荐页面。

主要修改文件：

src/pages/RecommendationView.vue

这个页面的目标是展示 AI 根据当前日期、用户日程、历史饮食、过敏信息和饮食偏好生成的饮食推荐。

你可能会用到这些文件：

src/stores/date.ts
src/stores/user.ts
src/api/recommendation.ts
src/types/index.ts

已有的日期状态：

useDateStore

里面提供：

selectedDate
selectedMonth
setSelectedDate()
goToday()

推荐接口已经封装在：

src/api/recommendation.ts

可以调用：

getRecommendation(date)

示例：

const { selectedDate } = storeToRefs(useDateStore())

const result = await getRecommendation(selectedDate.value)

推荐结果的数据类型是：

MealRecommendation

定义位置：

src/types/index.ts

建议开发内容：

1. 主要修改 src/pages/RecommendationView.vue
2. 从 src/stores/date.ts 读取当前选中的日期
3. 调用 getRecommendation(selectedDate.value)
4. 展示 AI 推荐的早餐、午餐、晚餐或加餐
5. 展示 AI 给出的推荐理由
6. 增加 loading 状态，比如“正在生成推荐”
7. 增加 error 状态，比如“推荐生成失败，请稍后重试”
8. 一般不要修改 src/api/request.ts
9. 一般不要修改 src/router/index.ts，除非需要新增页面

简单来说，你只需要重点改：

src/pages/RecommendationView.vue

如果要请求后端，就使用：

src/api/recommendation.ts

如果要读取当前日期，就使用：

src/stores/date.ts
